== Description ==

Add custom post types to the Millionaire's Digest for the pupose of allowing our users to create, upload, and share them with everyone, as well as for the pupose of allowing us to register them through Jetpack's plugin.


Features:-

Post types include:
* Articles.
* Videos.
* Photos.
* Music.
* Company.
* Magazine Articles.


== Installation ==

1. Add new
2. Upload plugin
3. Install plugin
4. Click activate
5. You're all set!
